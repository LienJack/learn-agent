1. 图解类型
环形飞轮图，因为可恢复、可评估、可委派会形成 Managed Agent 的持续改进闭环。

2. 画面元素清单
6 个节点：Session log、Replay、Eval、Harness Fix、Sub-agent Handoff、Production Run。高亮 Eval 和 Harness Fix。中心写 Managed Agent 控制核心。箭头形成顺时针飞轮。

3. 正向图片提示词
画一张文章内技术解释图，主题是“Managed Agent 通过回放、评估和委派形成改进飞轮”。

画风：米白色纸张背景，黑色手绘签字笔线稿，线条粗细略不均，少量淡黄色高亮，editorial technical illustration，技术博客手绘流程图，清晰、克制、有工程草图感。

构图：环形飞轮图。中央画 Managed Agent 控制核心：小控制台和护栏。外围顺时针排列 6 个节点：① Session log：事件时间线。② Replay：回放箭头。③ Eval：测试量尺和勾选，使用淡黄色高亮。④ Harness Fix：扳手修控制台，使用淡黄色高亮。⑤ Sub-agent Handoff：任务卡片交接。⑥ Production Run：项目文件夹和运行指示灯。节点之间用粗细略不均的手绘箭头连接成持续循环。

背景有很淡的电路线和工程草图辅助线。标签短，底部留出一行空白金句位置。整体像解释运维改进机制的技术博客手绘图。

4. 负向提示词
