ノード：
① 親タスク：小さな旗、テスト修復の主目的を表す。
② タスクパッケージ：封筒 + チェックリスト、親 Agent が出した scoped task を表す。
③ 子 Agent：小人物 + 虫眼鏡、局所調査を表す。
④ 構造化結果：証拠カード + 疑問符、淡い黄色で強調し、evidence と unknowns を表す。
⑤ Join ゲート：合流スイッチ、淡い黄色で強調し、親 Agent が証拠を統合することを表す。
