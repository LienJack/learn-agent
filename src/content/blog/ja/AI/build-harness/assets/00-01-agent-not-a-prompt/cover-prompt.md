## Prompt

「Agent はより長い Prompt ではなく、モデル、ループ、ツール、状態、制御システムで構成される実行プロセスである」というテーマの技術ブログカバー画像を描く。

構図：横長のブログカバー。左側に小さな入力カード「Prompt」を置き、短い英語ラベルのみを使う。中央には、単発の回答から制御された実行へ進む手描きの経路を置く。右側には、5 つの短いラベルのモジュール Model、Loop、Tools、State、Harness で構成される、より完全な Agent 実行装置を置く。矢印で「テキスト要求から制御可能な実行プロセスへの進化」を表現する。
