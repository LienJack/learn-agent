1. 図解タイプ
Layered architecture。Local Tool Bundle は関数リストではなく、モデル意図とローカル環境の間に挟まる制御された runtime 層であることを説明する。

2. 画面要素リスト
縦方向 5 層：モデル意図、ツール契約、検証と権限、ローカル実行器、ファイルシステムと端末。右側に細い監査チェーン。ツール契約と権限 runtime を強調する。
