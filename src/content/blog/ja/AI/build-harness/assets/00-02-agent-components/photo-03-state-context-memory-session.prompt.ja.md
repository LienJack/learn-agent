1. 図解タイプ
階層アーキテクチャ図。State、Context、Memory、Session log は、永続性と投影関係が異なる層であるため。

2. 画面要素リスト
4 層：Session log、State、Context、Memory。State が Session log から集約されること、Context が State からモデルへの一回ごとの投影であること、Memory が再利用可能な長期知識であることを矢印で表す。Context 投影と Session log の事実源を強調する。
