# Blog To Photo Section Prompt

Use this request with the `blog-to-photo` skill.

## Input

- Article path
- Section heading
- Section excerpt
- Target asset path

## Output

Return:

1. 图解类型
2. 画面元素清单
3. 正向图片提示词
4. 负向提示词

Keep labels short. Prefer technical blog hand-drawn illustration style. Avoid dense text, UI screenshots, 3D, photorealism, dark backgrounds, neon colors, and more than 8 main nodes.
